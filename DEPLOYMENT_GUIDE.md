# راهنمای قدم به قدم دیپلوی ربات تلگرام روی Railway از طریق GitHub

این راهنما به شما کمک می‌کند تا ربات تلگرام هوش مصنوعی خود را که با `python-telegram-bot` و `Google Gemini` توسعه داده شده است، به راحتی روی پلتفرم Railway دیپلوی کنید. این فرآیند شامل ایجاد یک ریپازیتوری در GitHub، اتصال آن به Railway و تنظیم متغیرهای محیطی است.

## پیش‌نیازها

قبل از شروع، مطمئن شوید که موارد زیر را در اختیار دارید:

1.  **حساب GitHub:** برای میزبانی کد ربات.
2.  **حساب Railway:** برای دیپلوی و اجرای ربات.
3.  **توکن ربات تلگرام:** (مثال: `YOUR_TELEGRAM_BOT_TOKEN_HERE`)
4.  **کلید API گوگل Gemini:** (مثال: `YOUR_GEMINI_API_KEY_HERE`)

## مراحل دیپلوی

### مرحله 1: ایجاد ریپازیتوری در GitHub

1.  وارد حساب GitHub خود شوید.
2.  یک ریپازیتوری جدید ایجاد کنید. می‌توانید نامی مانند `telegram-gemini-bot` برای آن انتخاب کنید. مطمئن شوید که ریپازیتوری **Public** یا **Private** باشد (Railway به هر دو متصل می‌شود).
3.  فایل‌های زیر را که توسط من آماده شده‌اند، به این ریپازیتوری پوش (Push) کنید:
    -   `bot.py` (کد اصلی ربات)
    -   `requirements.txt` (لیست کتابخانه‌های مورد نیاز)
    -   `Procfile` (دستور اجرای ربات برای Railway)
    -   `.gitignore` (فایل‌هایی که نباید در گیت قرار بگیرند)
    -   `README.md` (توضیحات پروژه)

    برای این کار، می‌توانید ریپازیتوری را کلون کرده، فایل‌ها را کپی کنید و سپس پوش کنید، یا مستقیماً از طریق رابط کاربری GitHub فایل‌ها را آپلود کنید.

### مرحله 2: اتصال GitHub به Railway

1.  وارد حساب Railway خود شوید.
2.  روی دکمه `New Project` کلیک کنید.
3.  گزینه `Deploy from GitHub Repo` را انتخاب کنید.
4.  اگر قبلاً GitHub خود را به Railway متصل نکرده‌اید، این کار را انجام دهید و به Railway اجازه دسترسی به ریپازیتوری‌های خود را بدهید.
5.  ریپازیتوری `telegram-gemini-bot` (یا هر نامی که انتخاب کرده‌اید) را از لیست ریپازیتوری‌های خود انتخاب کنید.
6.  Railway به طور خودکار پروژه شما را شناسایی کرده و شروع به دیپلوی می‌کند.

### مرحله 3: تنظیم متغیرهای محیطی (Environment Variables)

این مرحله بسیار مهم است، زیرا ربات برای کار کردن به توکن تلگرام و کلید Gemini نیاز دارد.

1.  پس از اینکه Railway شروع به دیپلوی پروژه کرد، وارد صفحه پروژه خود در Railway شوید.
2.  به بخش `Variables` بروید.
3.  روی `New Variable` کلیک کنید و دو متغیر زیر را اضافه کنید:
    -   **نام:** `TELEGRAM_BOT_TOKEN`
        **مقدار:** توکن ربات تلگرام خود را اینجا وارد کنید (مثال: `YOUR_TELEGRAM_BOT_TOKEN_HERE`)
    -   **نام:** `GEMINI_API_KEY`
        **مقدار:** کلید API گوگل Gemini خود را اینجا وارد کنید (مثال: `YOUR_GEMINI_API_KEY_HERE`)

    **توجه:** همانطور که قبلاً اشاره شد، کلید Gemini شما در تست‌های اولیه با مشکل مواجه شد. لطفاً از صحت و فعال بودن آن اطمینان حاصل کنید. در صورت بروز مشکل، ممکن است نیاز به بررسی تنظیمات Google Cloud یا ایجاد یک کلید جدید داشته باشید.

4.  پس از اضافه کردن متغیرها، Railway به طور خودکار پروژه را با متغیرهای جدید ری‌استارت و دیپلوی مجدد می‌کند.

### مرحله 4: بررسی وضعیت ربات و آنلاین شدن

1.  به بخش `Deployments` در صفحه پروژه Railway خود بروید.
2.  منتظر بمانید تا وضعیت دیپلوی به `Deployed` تغییر کند. این ممکن است چند دقیقه طول بکشد.
3.  می‌توانید لاگ‌های (Logs) ربات را در بخش `Logs` مشاهده کنید تا از صحت عملکرد آن مطمئن شوید. اگر خطایی مشاهده کردید، می‌توانید از لاگ‌ها برای عیب‌یابی استفاده کنید.
4.  پس از موفقیت‌آمیز بودن دیپلوی، ربات شما آنلاین خواهد شد و می‌توانید با آن در تلگرام ارتباط برقرار کنید.

حالا ربات شما آماده استفاده است! می‌توانید با ارسال پیام، عکس یا لینک یوتیوب، قابلیت‌های آن را تست کنید.
