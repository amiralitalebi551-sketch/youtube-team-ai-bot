# ربات تلگرام هوش مصنوعی برای تیم یوتیوب توسعه فردی

این ربات تلگرام با استفاده از `python-telegram-bot` و `Google Gemini` ساخته شده است تا به تیم ۳ نفره یوتیوب شما در حوزه توسعه فردی کمک کند. این ربات با لحنی صمیمی و محاوره‌ای فارسی پاسخ می‌دهد، اعضای تیم را با اسم می‌شناسد و قابلیت تحلیل عکس و لینک‌های یوتیوب را دارد.

## قابلیت‌ها

- **گفتگوی هوشمند و صمیمی:** با لحن فارسی محاوره‌ای و دوستانه با اعضای تیم ارتباط برقرار می‌کند.
- **شناسایی اعضای تیم:** جواد (@TheToll_man)، حمید (@Hadim848) و طالب (@Far_Boo) را با اسم می‌شناسد و با آن‌ها صمیمی برخورد می‌کند.
- **حافظه پایدار گفتگو:** تاریخچه گفتگوها را به صورت پایدار ذخیره می‌کند تا با ری‌استارت ربات پاک نشود.
- **تحلیل عکس:** قابلیت تحلیل تصاویر ارسالی را دارد.
- **تحلیل لینک یوتیوب:** با دریافت لینک ویدیو یوتیوب، محتوای آن را تحلیل کرده و نکات کلیدی یا ایده‌های مرتبط را ارائه می‌دهد.
- **مدیریت خطا:** دارای مکانیزم‌های اولیه مدیریت خطا برای پایداری بیشتر.
- **دستورات کاربردی:**
    - `/start`: شروع مکالمه و معرفی ربات.
    - `/reset`: پاک کردن حافظه گفتگوی فعلی.
    - `/help`: نمایش راهنمای استفاده از ربات.

## متغیرهای محیطی (Environment Variables)

برای اجرای ربات، نیاز به تنظیم متغیرهای محیطی زیر دارید:

- `TELEGRAM_BOT_TOKEN`: توکن ربات تلگرام شما.
- `GEMINI_API_KEY`: کلید API سرویس Google Gemini شما.

**نکته مهم:** کلید Gemini شما (با فرمت `YOUR_GEMINI_API_KEY_HERE`) در تست‌های اولیه با خطای `429 Quota Exceeded` مواجه شد. لطفاً از فعال بودن و داشتن سهمیه کافی برای این کلید اطمینان حاصل کنید. ممکن است نیاز به بررسی تنظیمات صورتحساب یا ایجاد یک کلید جدید داشته باشید.
